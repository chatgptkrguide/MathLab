# App Icon Generation Instructions

앱 아이콘 생성을 위해 다음 단계를 수행하세요:

1. 1024x1024 크기의 PNG 이미지 생성
   - 파란색 그라디언트 배경
   - 중앙에 π (파이) 기호
   - 하단에 작은 수식 (∫ f(x) dx)
   - 주변에 작은 수학 기호들 (∑, √, ∞)

2. 온라인 아이콘 생성기 사용
   - https://www.appicon.co/
   - https://icon.kitchen/

3. 또는 Figma/Canva로 디자인 후 이미지 생성

임시로 텍스트 기반 아이콘을 생성합니다.
